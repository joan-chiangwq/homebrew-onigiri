# homebrew-onigiri

Homebrew Cask tap for [Onigiri](https://github.com/joan-chiangwq/on-igiri) — the
smallest possible way to keep your Mac awake.

## Install

```sh
brew tap joan-chiangwq/onigiri
brew install --cask onigiri
```

## Updating

```sh
brew upgrade --cask onigiri
```

This tap is updated as part of `joan-chiangwq/on-igiri`'s release process —
see that repo's `distribution/README.md`.
